# Adding Your Profile Photo and Project Images

Since the image generation service is temporarily unavailable, here's how to add your own images:

## Profile Photo

1. **Find or take a photo** of yourself (square format works best)
2. **Save it as** `photo.jpg` in the `public/assets/profile/` folder
3. **Recommended size**: 300x300px or larger (it will be displayed at 150x150px)

## Project Images

For each project, create a folder in `public/assets/projects/` and add:

### Soulcare Dynamics (`public/assets/projects/soulcare/`)
- Take screenshots of the live website
- Save as `thumb.jpg`, `homepage.jpg`, `pricing.jpg`, etc.

### MapGame (`public/assets/projects/mapgame/`)
- Screenshots of the game dashboard
- Save as `thumb.jpg`, `dashboard.jpg`, etc.

### Chad AI (`public/assets/projects/chad-ai/`)
- Screenshots of the AI interface
- Save as `thumb.jpg`, `interface.jpg`, etc.

### School Projects (`public/assets/projects/school/`)
- Screenshots or collage of your school work
- Save as `thumb.jpg`, `collection.jpg`, etc.

## Quick Tips

- **Thumbnails** should be 16:9 or 5:3 aspect ratio for best display
- **File formats**: JPG, PNG, or WebP
- **Optimization**: Compress images to keep file sizes small (use tools like TinyPNG)
- **Naming**: Use descriptive names without spaces (use hyphens instead)

## Temporary Placeholders

Until you add real images, the site will show:
- Colored letter placeholders for project thumbnails
- A broken image icon for the profile photo (which you can replace)

The site is fully functional even without images - they just enhance the visual appeal!
